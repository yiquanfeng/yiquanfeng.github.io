# yiquanfeng.github.io

一个可直接部署到 GitHub Pages 的个人作品展示静态站点。

## 文件说明

- `index.html`：页面结构与文案
- `styles.css`：页面样式、响应式布局和动效
- `script.js`：滚动出现动画

## 使用方式

1. 直接修改 `index.html` 里的个人介绍、作品名称、链接和联系方式。
2. 如果要替换视觉风格，优先调整 `styles.css` 顶部的颜色变量。
3. 提交并推送到 `main` 分支后，仓库名是 `yiquanfeng.github.io`，可直接作为 GitHub Pages 主页仓库使用。

## 建议下一步

- 把作品卡片换成你的真实项目
- 把 `your-email@example.com` 替换成真实邮箱
- 给每个项目补上线上演示链接或截图
